# 給食管理システム

C# Windows Forms で作る給食管理システムです。

## 名簿読み込み

Excel の名簿は、Excel で「CSV UTF-8」に保存してから読み込みます。

CSV に必要な列は以下の4つです。

- `学年`
- `組`
- `番号`
- `氏名`

区分も一緒に読み込む場合は、先頭に `区分` 列を追加します。
指定できる値は `職員`、`生徒`、`ALT`、`教育実習生`、`試食会`、`ゲスト` です。
`区分` 列がない場合は、これまでどおり生徒として読み込みます。
`開始日` 列を追加すると、`2026/4/1` のような日付を在籍開始日として読み込みます。

全区分を試すサンプルは `samples\区分全員テスト.csv` にあります。
100人分のランダムなサンプルは `samples\区分全員ランダム100人.csv` にあります。

`氏名` は、姓と名の間に空白がある場合は分けて読み込みます。空白がない場合は、いったん姓にまとめて読み込みます。

CSVから読み込んだ人は、区分は `生徒`、喫食日は月〜金、牛乳あり、アレルギー対応なしで登録されます。配膳場所、牛乳、アレルギーなどは、読み込み後に編集画面で調整します。

CSVから読み込んだ生徒は、`学年` と `組` から `1年1組` のような配膳場所が自動登録され、初期の配膳場所履歴にも登録されます。月途中や年度途中で配膳場所が変わる場合は、編集画面の `履歴管理` から開始日・終了日付きで登録します。

転入、職員、ALT、教育実習生、試食会、ゲストなどの追加は、画面の「1人追加」から登録します。

## 保存場所

データは以下に保存します。

`Documents\KyushokuKanriSystem\data.json`

## 月別給食数

`月別給食数` 画面では、対象月を選ぶと、日付・配膳場所ごとの提供数、牛乳数、
アレルギー対応数、停止・欠席数を一覧で確認できます。
月途中で配膳場所が変わる場合も、その日の配膳場所履歴を使って集計します。
月全体はカレンダー形式で表示され、日付を選ぶと配膳場所別の内訳を確認できます。
停止・欠席数の `確認` ボタンから、該当者の氏名、状態、理由を確認できます。
`月間一覧` タブでは、学級・区分を縦軸、1日から31日を横軸にした一覧表で、
日ごとの給食数と月合計を比較できます。
